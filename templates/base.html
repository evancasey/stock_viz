Visualizing the "Stock Tweets" Community with D3

Followers Retweets Mentions
A visualization of different user influence measures across the top 1% most influential $TICKER users. Data collected from the Twitter Streaming API between 4/22/13 and 4/29/13.

Mouse over each node for more information on a particular user. Bubble size indicates the normalized follow/mention/retweet count for each user. You can also drag and drop the nodes, go ahead...it's pretty fun.

Built using D3.
